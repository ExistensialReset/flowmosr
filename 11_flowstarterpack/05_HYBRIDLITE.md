# 🧩 Hybrid Survival Guide (Condensed)

For people who cannot “just leave.”

## PHASE 0 — Awareness
Track:
- money spent this month 
- hours worked vs hours resting
- stress levels (0–10) at key points 

## PHASE 1 — Reduce Bleeding
Cut one parasitic cost each month.
Build a €/$/SEK 500–1000 emergency buffer if possible OR a store of essentials (food, water, supplies).

## PHASE 2 — Parallel Infrastructure
Share tools, bulk-buy food, start a micro-garden.
Reduce work hours gradually - even one hour less per week is progress. 

## PHASE 3 — Threshold Test
Simulate one month without income.
If Circle + savings cover 50%+ of basic needs → hybrid is viable.

## PHASE 4 — Hybrid Permanence
Work 2–4 days/week in Mammon.
Live the rest in Flow.

**Hybrid is not failure.  
Hybrid is survival.**
