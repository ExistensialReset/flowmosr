# 🚀 First 7 Days — Start Flow Without Calling It Flow

**Day 1**: Find 2–5 people. Ask: “What is hardest right now?”  
**Day 2**: Choose ONE small shared action (meals, water, childcare, tools).  
**Day 3**: Do it once.  
**Day 4**: Repeat + adjust.  
**Day 5**: Add a second small action.  
**Day 6**: Create a simple rotation.  
**Day 7**: Sit together and reflect: What became easier?

**If people feel**:
- **Less** stress  
- **More** time  
- **More support**  

They will continue.  
This is Level 0.
