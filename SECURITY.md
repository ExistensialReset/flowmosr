# Security Policy

This repository is public. Do not post personal data, secrets, or sensitive evidence in issues, pull requests, or comments.

## Reporting security issues
If you discover a security vulnerability or need to report sensitive information, do **not** open a public issue. Instead contact the project maintainer privately:

Email: elinor.frejd@proton.me  
Subject: Security report for Existential Reset — [short description]

Include: brief summary, affected files or URLs, and steps to reproduce. Do not include personal data or attachments in the initial message.

## Response timeline
We will acknowledge receipt within 48 hours and provide an estimated timeline for triage and remediation.

## Responsible disclosure
We follow a responsible disclosure approach. We will coordinate fixes and public disclosure with the reporter where appropriate.

## Public repo notice
This repository is intentionally public. Any content posted publicly may be visible to anyone. Do not post PII, medical records, or other sensitive material here.
