# FLOW ANTI-GAMING LAYER (AGL)

## 1. OUTCOME BALANCE CHECK (over time)
The system evaluates fairness over time, not per decision.

If any actor consistently receives disproportionate advantage:
→ triggers review of allocation patterns, not individual cases

---

## 2. STRATEGIC CONSISTENCY DETECTION
If an actor repeatedly:
- maximizes benefit from CRISIS thresholds
- uses CEPP threats strategically
- benefits unusually from RESOURCE simulations

→ system flags "strategic exploitation pattern"

---

## 3. CEPP ANTI-WEAPONIZATION RULE
CEPP (exit rights) cannot be used as:
- bargaining leverage
- implicit veto pressure
- repeated negotiation tool

If detected:
→ CEPP still valid, but loses procedural influence weight in RESOURCE decisions

---

## 4. DISTRIBUTION ROTATION PRINCIPLE
Over time:
- no actor may remain in top-benefit position continuously
- system enforces natural rotation of advantage

Not punishment — just rebalancing

---

## 5. INTERPRETATION BIAS CORRECTION
If simulation / dialogue outcomes consistently favor:
- high verbal skill
- high strategic framing ability

→ system applies correction weight toward underrepresented needs

---

## 6. META-FAIRNESS LAYER
Final metric is not per decision fairness, but:
- long-term equity
- distribution stability
- participation balance

All reviewed periodically, not in real time conflict mode
