# `/guides/activation` — Activation Guides & Flow Implementation ⚡💚

**Status:** Draft  
**Scope:** All Flow Node participants interested in activation processes, ecological principles, and practical node workflows.  
**Purpose:** Central hub for activation-related guides, protocols, and practical resources. Designed for clarity, traceability, and operational alignment.  

---

## Overview

This folder contains **all activation-related guides**: from node governance, flow starter procedures, somatic checks, to ecological and divine principles. These guides are meant to **support node members in starting, maintaining, and scaling Flow operations safely and effectively**.  

**Key principles:**
- Node safety and ethical alignment first 🛡️  
- Clear, actionable workflows for activation and implementation 📝  
- Integration with ecological and divine frameworks 🌿✨  
- Documentation and traceability for all activation steps 🔗  

---

### Flow Initiation

- `START_SMALL.md` — Guidance for incremental Flow launches

- `FLOW_GOVERNANCE.md` — Node governance structure, responsibilities, and processes

- `DIVINE_STATE_OF_MIND.md` — Guidance on mindset and intention tracking

### Somatic & Neurodivergent Practices

- `NEURODIVERGENT_FLOW_GUIDE.md` — Adapted activation guidance for neurodivergent participants  

### Physics & Ecology
