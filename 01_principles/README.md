# 🌱 Flow Principles
## How life is protected in practice


If the core defines what must be true,  
these principles define **how we act in the world**.
For justice principles, see 06_justice, for economics, see 09_economics

---

## 🧭 What this is

Principles are not ideology.

They are **operational constraints** that ensure:
- Life is not harmed
- Power does not accumulate
- Systems remain adaptive

---

## ⚖️ Core direction

All principles follow one condition:

> Systems must serve life — or they must change.

---

## 📂 What you will find here

This folder contains principles for:

- Education
- Healthcare
- Childen's partipation
- Social structures

Each document answers:

**“What must remain true when this system is applied?”**

---

## 🧠 How to read this

You don’t need to read everything.

Start where you feel tension:
- If you question anything, read about it    

---

## 🌊 Important

Principles are not fixed forever.

But they are not easily changed either.

They exist to **prevent drift back into systems that harm life**.

---

## 💙 A small reminder

You don’t need perfect understanding.

You only need to ask:

> Does this increase or reduce unnecessary suffering?
